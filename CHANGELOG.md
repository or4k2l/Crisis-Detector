# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-17

### Added
- Initial release of Crisis Detector
- Multi-domain crisis detection framework
- Support for finance, seismic, gravitational wave, and EEG data
- Statistical and machine learning-based anomaly detection
- Comprehensive visualization tools
- Optional dependency management for domain-specific features
- Extensive test suite with basic and advanced tests
- Logging system for better debugging and monitoring
- CI/CD pipeline with GitHub Actions

### Features
- Z-score based anomaly detection
- Isolation Forest machine learning method
- Volatility and rate-of-change analysis
- Sliding window approach for time-series analysis
- Configurable detection parameters
- Multi-panel visualization plots

### Documentation
- Comprehensive README with examples
- API reference and usage guidelines
- Installation instructions with optional dependencies
- Contributing guidelines
- MIT License

### Technical Improvements
- Replaced print statements with proper logging
- Added optional dependencies for lighter installation
- Enhanced type hints for better code clarity
- Documented Isolation Forest 100-point threshold
- Improved error handling with detailed logging
